# line-counter



## Getting started

Run this to allow GitPython to run anywhere in your user environment. 
```sh
pip3 install -r requirements.txt
```

Otherwise, setup the venv with
```sh
./setup_venv.sh
source ./venv/bin/activate
```

## Running the program

```sh
python3 line-counter [Directory-or-Git-Link]
```

Git link can be ssh or https. On server, likely only ssh works.
